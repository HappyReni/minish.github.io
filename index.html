핵심 기능

미니시가 제공하는 특별한 경험

🔒

완벽한 개인정보 보호

모든 데이터는 로컬에 저장됩니다. 클라우드 동기화나 데이터 수집이 없어 완전한 프라이버시를 보장합니다.

🎨

미니멀한 디자인

깔끔하고 직관적인 UI/UX로 복잡한 설정 없이 바로 사용할 수 있습니다. 에메랄드/틸 컬러로 차분한 느낌을 제공합니다.

⚡

즉각적인 피드백

루틴 완료 시 시각적, 청각적, 햅틱 피드백을 통해 즉각적인 성취감과 만족감을 제공합니다.

📊

GitHub 스타일 히트맵

일일 루틴 달성률을 GitHub 잔디밭 스타일의 히트맵으로 시각화하여 진행 상황을 한눈에 파악할 수 있습니다.

📤

데이터 내보내기

CSV, JSON, Markdown 형식으로 데이터를 내보낼 수 있어 다른 도구와 연동하거나 백업이 가능합니다.

🌙

일일 마무리 & 기분 기록

하루를 마무리하며 완료율을 확인하고 기분을 기록할 수 있습니다. 기분 변화를 차트로 추적할 수 있습니다.

개인정보 보호 정책

투명하고 안전한 데이터 처리

로컬 저장 우선

모든 루틴 데이터와 통계는 사용자 기기에만 저장됩니다. 외부 서버로 전송되지 않습니다.

최소한의 권한

앱 실행에 꼭 필요한 권한만 요청합니다. 불필요한 개인정보 접근은 하지 않습니다.

광고 정책

앱 개발 지원을 위한 최소한의 광고가 포함되어 있습니다. Google AdMob을 통해 제공됩니다.

데이터 내보내기

언제든지 데이터를 내보낼 수 있으며, 앱을 삭제하면 모든 데이터가 완전히 제거됩니다.
